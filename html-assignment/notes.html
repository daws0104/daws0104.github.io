Html tags Description
<Div> short for divide. It divides the child elements into different lines.
<Span> is for small content. It keeps the keeps the content on the same line as other content.
<Ol> stands for ordered list. For a list where item order does matter.
<Ul> stands for Unordered list. For a list where item order does not matter.
<Em> is short for emphasis. For emphasising things in Italics. like used above.
